ЛАБОРАТОРНАЯ РАБОТА №4 ДЕ ПОЙ ИДБ-24-14
============================================================

Это программа заключает в себе функционал бота, который передаётся по токену в телеграм-бот. В этом боте прописан вариант космоса (12) и прописаны команды: запуска работы бота, показа фото земли со спутника, фото дня от NASA, справки о планетах и новостей о космосе.
