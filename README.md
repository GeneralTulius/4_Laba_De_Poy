ЛАБОРАТОРНАЯ РАБОТА №4 ДЕ ПОЙ ИДБ-24-14
============================================================

Это программа заключает в себе функционал бота, который работает по уникальному токену и API от NASA. В этом боте прописан вариант космоса (12) и прописаны команды: запуска работы бота, показа фото земли со спутника, фото дня от NASA, справки о планетах, новостей о космосе, показа всех команд и смены языка у бота. Эти команды можно соответственно вызвать написав: /start или 1, /apod или 2, /planets или 3, /news или 4, /help и /set_lang. Так как бот работает с API NASA, в коде используется прокси для обхода блокировки. По желанию можно вставить вместо DEMO_KEY свой API с api.nasa.gov.
